# Portafolio Elvis Chávez

Portfolio personal con CV interactivo, proyectos y más.

## Stack

- React 18 + Vite 5
- Bootstrap 5 (CDN)
- React Router v6
- Google Material Icons

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Conectado a Netlify — push a main despliega automáticamente.
